🎮 Hangman Game (Python GUI)

 Description:
This is a GUI-based Hangman game developed using Python and Tkinter.

 Features:
- Random word selection
- Hint system
- Keyboard input support
- Animated hangman drawing
- Restart option

 Technologies Used:
- Python
- Tkinter

 How to Run:
1. Install Python
2. Run:
   python hangman.py
